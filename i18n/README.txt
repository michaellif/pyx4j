 ========= I18n =========
 Build should run our of the box on Linux.

 For the build on windows you need to have a gettext installed.
   Binary installer for gnu gettext 0.17 is available here http://www.cubicweb.org/blogentry/565331
     http://ftp.logilab.org/pub/gettext/gettext-0.17-win32-setup.exe
     http://pyx4j.com/downloads/gettext-0.17-win32-setup.exe

 There are many tools to maintain and Edit the .po files.
  See http://www.gnu.org/software/gettext/

  Nice tool to run on Windows/Mac: http://www.poedit.net/

  Online service for collaborative translate .po files:  https://www.transifex.net/
