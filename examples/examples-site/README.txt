============
Installation
============

1. Install Java
	Current version 1.6.0_17

2. Install Eclipse
	Current version - eclipse-rcp-galileo-SR1-win32
	 
3. Install Eclipse plug-ins
	- subclipse http://subclipse.tigris.org/update_1.6.x
	- m2eclipse http://m2eclipse.sonatype.org/update/
	- google plugin http://dl.google.com/eclipse/plugin/3.5
	
4. Set variable COLLAGE to collage modules root
